## Purpose

Track objective release evidence used for store-review and compliance verification of randomized pack disclosures.

## Required evidence per release

1. Openability audit output.
2. Sample API responses for:
   - `GET /api/products` (pack disclosure fields)
   - `GET /api/packs/{id}/odds`
3. Seed/backfill execution trace for pack data.
4. Automated test evidence for odds/openability behavior.
