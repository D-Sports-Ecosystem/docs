## High-level flow

```mermaid
flowchart LR
  packBrowse[PackBrowse] --> packPurchase[PackPurchase]
  packPurchase --> purchaseRecord[PackPurchaseRecord]
  purchaseRecord --> packOpen[PackOpenAction]
  packOpen --> collectibleAward[UserCollectibleAward]
  collectibleAward --> mintPath[OptionalMintPersistence]
  collectibleAward --> inventoryRead[InventoryAndOwnershipReads]
```

## Architectural notes

- Pack open behavior resolves weighted contents and persists ownership outcomes.
- Collectible ownership and mint records are related but not identical entities.
- Featured collectible views are read-optimized on top of core ownership models.
