Set up Claude Code to work on the D-Sports docs (this repo). Use the Mintlify skill for components and style, and [AGENTS.md](https://github.com/d-sports/docs/blob/main/AGENTS.md) for D-Sports-specific guidance.

## Prerequisites

- Active Claude subscription (Pro, Max, or API access)

## Setup

<Steps>
  <Step title="Install Claude Code">
    ```bash
    npm install -g @anthropic-ai/claude-code
    ```
  </Step>
  <Step title="Open the D-Sports docs repo and install the Mintlify skill">
    In the docs repo root, run:
    ```bash
    npx skills add https://mintlify.com/docs
    ```
    This gives Claude Code Mintlify's component reference and writing standards for [docs.d-sports.org](https://docs.d-sports.org).
  </Step>
  <Step title="Use AGENTS.md">
    The repo's `AGENTS.md` defines D-Sports terminology, style, and a Mintlify MDX quick reference. Keep it in mind when asking Claude to edit docs.
  </Step>
  <Step title="Start writing">
    Run `claude` in the repo and ask it to edit or create MDX pages. Preview with `mint dev`.
  </Step>
</Steps>
